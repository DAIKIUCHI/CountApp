ログログ　プライバシーポリシー

このアプリは、アプリ内で入力された情報を記録します。

入力した数値をアプリ内に記録します。

記録しておかないと次にアプリを立ち上げた時に、入力した数値が消えてしまいます。

そのために、入力した数値をアプリに記録させる機能があります。

アプリが入手する情報

  • 何をするか（「1日の摂取カロリーを決める」など）
  • 入力した数値
  • 端末情報

アプリが入手しない情報

以下のような個人情報（個人を特定できる情報）は一切入手しませんし、入手できません

  • 名前
  • 生年月日
  • 住所
  • 電話番号
  • メールアドレス
  • クレジットカード情報
